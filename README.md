



# Localiza-URI

O projeto Localiza URI tem como objetivo ajudar os alunos por meio de um mapa da universidade mostrando os principais pontos do Campus.
## Instalação

Instale Localiza URI com npm

Primeiro instale o Node.js: 
```bash
https://nodejs.org/en/
```
```bash
  cd Localiza-Uri
  npm install
  npx expo start

```
